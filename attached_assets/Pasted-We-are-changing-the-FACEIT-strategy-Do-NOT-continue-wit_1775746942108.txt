- FACEIT OAuth login
- load the user's FACEIT matches inside the app

Why we are doing this:
- much simpler
- less fragile
- less API/legal risk
- no need to contact FACEIT for private download API access
- still gives a very good user experience

Target flow:
1. User logs into FACEIT via OAuth

3. App shows match info (map, result, date, etc.)
4. App has a button like:
   - "Auf FACEIT öffnen"

5. User downloads the demo on FACEIT manually
6. User returns to the app
7. App scans the Downloads folder
8. App detects new .dem / .dem.gz / .dem.zst files
9. App extracts/moves the demo into the replay folder
10. App enables command generation / voice modes / CS2 opening
